Python Datatypes 

 

Variables can hold values and every value has a data type.python is a dynamically typed languages so there is  no need to declare its type

 

 

Python support following data types:

  • Numbers
  • Sequence
  • Sets
  • None
  • Mappings

 

Python flexibility with datatypes makes it powerfull and versatile, allowing you to focus on solving problems rather than getting baogged down by type related complexities.
