File Operations

  • File operations are essential for reading from and writing to files. Python provides a simple and powerful set of functions for working with files.
  • Python provides specialized libraries like csv, json, and others.

Here are some common file operations:

Opening a file 

  • To open a file, you can use the built-in open() function.
  • "filename.txt" is the name of the file you want to open.
  • "mode" specifies the mode in which the file is opened (e.g., read, write, append).
  • "r": Read (default mode)
  • "w": Write (creates a new file or truncates an existing file)
  • "a": Append (opens the file for writing at the end of the file)
  • "b": Binary mode (e.g., "rb", "wb", "ab")
try:
         # Specify the file path and mode ('r' for reading in this example)
         file_path = "example.txt"
     
         # Open the file for reading
         with open(file_path, 'r') as file:
             # Read the contents of the file
             file_contents = file.read()
     
             # Display the contents
             print("File Contents:")
             print(file_contents)
             # Output: File Contents: <contents of the file>
     
     except FileNotFoundError:
         # Handle FileNotFoundError
         print(f"Error: File '{file_path}' not found.")
         # Output: Error: File 'example.txt' not found.
     
     except Exception as e:
         # Handle any other unexpected errors
         print(f"An unexpected error occurred: {e}")
         # Output: An unexpected error occurred: <error message>

Reading from a file 

  • To read from a file, you can use methods like read(), readline(), or readlines()
try:
         # Specify the file path and mode ('r' for reading in this example)
         file_path = "example.txt"
     
         # Open the file for reading
         with open(file_path, 'r') as file:
             # Read the entire contents of the file into a string
             file_contents = file.read()
     
             # Display the contents
             print("File Contents:")
             print(file_contents)
             # Output: File Contents: <contents of the file>
     
             # Alternatively, you can read line by line
             print("\nFile Contents Line by Line:")
             file.seek(0)  # Reset the file pointer to the beginning
             for line in file:
                 print(line.strip())
     
             # Output: File Contents Line by Line:
             # <line 1 of the file>
             # <line 2 of the file>
             # ...
     
     except FileNotFoundError:
         # Handle FileNotFoundError
         print(f"Error: File '{file_path}' not found.")
         # Output: Error: File 'example.txt' not found.
     
     except Exception as e:
         # Handle any other unexpected errors
         print(f"An unexpected error occurred: {e}")
         # Output: An unexpected error occurred: <error message>

Writing to a file 

  • To write to a file, you can use the write() method:
try:
         # Specify the file path and mode ('w' for writing in this example)
         file_path = "output.txt"
     
         # Open the file for writing
         with open(file_path, 'w') as file:
             # Write content to the file
             file.write("Hello, this is a line written to the file.\n")
             file.write("This is another line.\n")
             # You can also use writelines() to write a list of lines
     
         print("File has been successfully written.")
         # Output: File has been successfully written.
     
     except Exception as e:
         # Handle any unexpected errors
         print(f"An unexpected error occurred: {e}")
         # Output: An unexpected error occurred: <error message>

Closing a file 

  • It's important to close the file after performing operations to free up system resources. You can use the close() method
try:
         # Specify the file path and mode ('r' for reading in this example)
         file_path = "example.txt"
     
         # Open the file for reading
         with open(file_path, 'r') as file:
             # Read the entire contents of the file into a string
             file_contents = file.read()
     
             # Display the contents
             print("File Contents:")
             print(file_contents)
             # Output: File Contents: <contents of the file>
     
         # The file is automatically closed when exiting the 'with' block
     
     except FileNotFoundError:
         # Handle FileNotFoundError
         print(f"Error: File '{file_path}' not found.")
         # Output: Error: File 'example.txt' not found.
     
     except Exception as e:
         # Handle any other unexpected errors
         print(f"An unexpected error occurred: {e}")
         # Output: An unexpected error occurred: <error message>

Using with Statement

  • To ensure that a file is properly closed after usage, you can use the with statement. This is a recommended practice as it automatically takes care of closing the file:
try:
         # Specify the file path and mode ('r' for reading, 'w' for writing in this example)
         read_file_path = "example.txt"
         write_file_path = "output.txt"
     
         # Reading from a file using 'with'
         with open(read_file_path, 'r') as read_file:
             # Read the entire contents of the file into a string
             file_contents = read_file.read()
     
             # Display the contents
             print("File Contents:")
             print(file_contents)
             # Output: File Contents: <contents of the file>
     
         # Writing to a file using 'with'
         with open(write_file_path, 'w') as write_file:
             # Write content to the file
             write_file.write("Hello, this is a line written to the file.\n")
             write_file.write("This is another line.\n")
             # You can also use writelines() to write a list of lines
     
         print("File has been successfully written.")
         # Output: File has been successfully written.
     
     except FileNotFoundError:
         # Handle FileNotFoundError
         print(f"Error: File not found.")
     
     except Exception as e:
         # Handle any other unexpected errors
         print(f"An unexpected error occurred: {e}")
         # Output: An unexpected error occurred: <error message>
