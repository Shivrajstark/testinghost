Relational Operators

These operators are used to create logical relationships between two values. Operators are used for comparison purposes. A relational operator is called a relational expression. Mainly relational operators are categorized into two sections, one is comparison and second is for equality. There are mainly 6 relational operators used in a language. First four are used for comparison purposes and the last two are used in equality sections. So the name of them are

less than,

less than, equal to

greater than,

greater than, equals to,
equal to and

not equal to.

 

Logical Operators

Logical operators are used for logical operations. These operators are used for compound relational expression. These are used in decision making statements and some looping statements. If, switch, do while these statement either have true or false block

 

Assignment Operators

 assignment operator are used for assigning an expression or value. Assignment operators are further subdivided into two types: simple assignment and shorthand assignment. Conditional Operator Conditional operator is a very powerful and convenient operator that can be used to replace certain elements of condition. Form conditional operators are also called ternary operators or, operators.

Identity Operators

In python , is and is not are the identity operators both are used to check if two values are located on the same part of the memory. Two variables that are equal do not imply that they are identical

is          True if the operands are identical

is not      True if the operands are not identical

 

Membership Operators

 

Python offers two membership operators to check or validate the membership of a value. It tests for membership in a sequence, such as strings, lists, or tuples. 

 

In Operators

The ‘in’ operator is used to check if a character/ substring/ element exists in a sequence or not. Evaluate to True if it finds the specified element in a sequence otherwise False.
