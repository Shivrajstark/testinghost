Operators

Operators in general are used to perform operations on values and variables. These are standard symbols used for the purpose of logical and arithmetic operations

For example a+b

Here,a and b are operands and +is operator

Basically operators are used to perform calculations

Arithmetic operators

arithmetic operators are used for arithmetical operations like addition, subtraction, multiplication, division etc. Mostly arithmetic operations. After are used in all computer languages. Arithmetic operators are designed at a time of the Abacus computing machine. There are mainly five types of operators that are used in the Python language.

Example suppose a=5 ,b=2

Operator       meaning              result

A+b                addition                7

A_b              subtraction              3

a*b               multiplication          10

a/b               division                   2

a%b            modulus                   1

 

# Simple Arithmetic Operators Example with Identifiers a and b
     
     # Define values for a and b
     a = 5
     b = 3
     
     # Addition
     sum_result = a + b
     print("Sum:", sum_result)  # Output: Sum: 8
     
     # Subtraction
     difference_result = a - b
     print("Difference:", difference_result)  # Output: Difference: 2
     
     # Multiplication
     product_result = a * b
     print("Product:", product_result)  # Output: Product: 15
     
     # Division
     quotient_result = a / b
     print("Quotient:", quotient_result)  # Output: Quotient: 1.6666666666666667
     
     # Floor Division
     floor_result = a // b
     print("Floor Result:", floor_result)  # Output: Floor Result: 1
     
     # Modulo (Remainder)
     remainder_result = a % b
     print("Remainder:", remainder_result)  # Output: Remainder: 2
     
     # Exponentiation
     power_result = a ** b
     print("Power:", power_result)  # Output: Power: 125

Relational Operators

These operators are used to create logical relationships between two values. Operators are used for comparison purposes. A relational operator is called a relational expression. Mainly relational operators are categorized into two sections, one is comparison and second is for equality. There are mainly 6 relational operators used in a language. First four are used for comparison purposes and the last two are used in equality sections. So the name of them are

less than,

less than, equal to

greater than,

greater than, equals to,
equal to and

not equal to.

# Relational Operators Example
     
     # Define values for a and b
     a = 10
     b = 5
     
     # Equal to
     equal_result = a == b
     print("Equal to:", equal_result)  # Output: Equal to: False
     
     # Not equal to
     not_equal_result = a != b
     print("Not Equal to:", not_equal_result)  # Output: Not Equal to: True
     
     # Greater than
     greater_than_result = a > b
     print("Greater Than:", greater_than_result)  # Output: Greater Than: True
     
     # Less than
     less_than_result = a < b
     print("Less Than:", less_than_result)  # Output: Less Than: False
     
     # Greater than or equal to
     greater_than_equal_result = a >= b
     print("Greater Than or Equal To:", greater_than_equal_result)  # Output: Greater Than or Equal To: True
     
     # Less than or equal to
     less_than_equal_result = a <= b
     print("Less Than or Equal To:", less_than_equal_result)  # Output: Less Than or Equal To: False

Logical Operators

Logical operators are used for logical operations. These operators are used for compound relational expression. These are used in decision making statements and some looping statements. If, switch, do while these statement either have true or false block

# Logical Operators Example with Values
     
     # Define values for x and y
     x = 10
     y = 5
     
     # Logical AND
     and_result = (x > 5) and (y < 10)
     print(f"Logical AND: {and_result}")  # Output: Logical AND: True
     
     # Logical OR
     or_result = (x < 5) or (y > 10)
     print(f"Logical OR: {or_result}")  # Output: Logical OR: False
     
     # Logical NOT
     not_result_x = not (x == 10)
     not_result_y = not (y != 5)
     print(f"Logical NOT x: {not_result_x}")  # Output: Logical NOT x: False
     print(f"Logical NOT y: {not_result_y}")  # Output: Logical NOT y: False

Assignment Operators

 assignment operator are used for assigning an expression or value. Assignment operators are further subdivided into two types: simple assignment and shorthand assignment. Conditional Operator Conditional operator is a very powerful and convenient operator that can be used to replace certain elements of condition. Form conditional operators are also called ternary operators or, operators.

     # Modulo assignment

     # Exponentiation assignment

Membership Operators

Python offers two membership operators to check or validate the membership of a value. It tests for membership in a sequence, such as strings, lists, or tuples. 

# Membership Operators Example
     
     # Define a list
     fruits = ["apple", "banana", "orange", "grape"]
     
     # Membership test using 'in'
     is_banana_in_list = "banana" in fruits
     is_mango_in_list = "mango" in fruits
     
     print("Is 'banana' in the list?", is_banana_in_list)  # Output: Is 'banana' in the list? True
     print("Is 'mango' in the list?", is_mango_in_list)    # Output: Is 'mango' in the list? False

In Operators

The ‘in’ operator is used to check if a character/ substring/ element exists in a sequence or not. Evaluate to True if it finds the specified element in a sequence otherwise False.

# Membership Operators (in and not in) Example
     
     # Define a list
     colors = ["red", "green", "blue", "yellow"]
     
     # Membership test using 'in'
     is_green_in_list = "green" in colors
     is_purple_in_list = "purple" in colors
     
     print("Is 'green' in the list?", is_green_in_list)   # Output: Is 'green' in the list? True
     print("Is 'purple' in the list?", is_purple_in_list)  # Output: Is 'purple' in the list? False
     
     # Membership test using 'not in'
     is_orange_not_in_list = "orange" not in colors
     is_blue_not_in_list = "blue" not in colors
     
     print("Is 'orange' not in the list?", is_orange_not_in_list)  # Output: Is 'orange' not in the list? True
     print("Is 'blue' not in the list?", is_blue_not_in_list)      # Output: Is 'blue' not in the list? False
