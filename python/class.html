Object Oriented Programming

Class

  • A class is a fundamental concept in object-oriented programming (OOP).
  • It acts as a blueprint for creating objects, providing a structure for their attributes and methods.
  • Classes enable the organization of code into logical units, promoting modularity and maintainability.

 

class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name} says Woof!")

# Creating an instance of the Dog class
my_dog = Dog(name="Buddy", age=3)

# Accessing attributes and calling methods
print(f"My dog's name is {my_dog.name}.")
# Output: My dog's name is Buddy.

print(f"My dog is {my_dog.age} years old.")
# Output: My dog is 3 years old.

my_dog.bark()
# Output: Buddy says Woof!
