Directory

  • A directory is referred to as a folder, and the term used in programming is a "directory" or "folder" interchangeably.
  • Directories are used to organize files into a hierarchical structure.
  • Python provides modules to interact with directories and perform various operations.

Working with Directories

  • The os module in Python is commonly used for interacting with the operating system, including working with directories.

Creating a Directory

  • You can create a new directory using the os.makedirs() function
import os
     
     # Specify the directory path
     directory_path = "/path/to/your/directory"
     
     # Create the directory
     os.makedirs(directory_path)

Checking if Directory exists

  • To check if a directory exists, you can use os.path.exists()
import os
     
     # Specify the directory path
     directory_path = "/path/to/your/directory"
     
     # Check if the directory exists
     if os.path.exists(directory_path) and os.path.isdir(directory_path):
         print(f"The directory '{directory_path}' exists.")
     else:
         print(f"The directory '{directory_path}' does not exist.")

Changing with current working Directory

  • You can change the current working directory using os.chdir()
import os
     
     # Specify the directory path
     new_directory_path = "/path/to/your/new/directory"
     
     # Change the current working directory
     os.chdir(new_directory_path)
     
     # Print the new current working directory
     print("Current working directory:", os.getcwd())

Listing Files and Directories 

  • To list the contents of a directory, you can use os.listdir()
import os
     
     # Specify the directory path
     directory_path = "/path/to/your/directory"
     
     # List files and directories in the specified directory
     file_list = os.listdir(directory_path)
     
     # Print the list
     print("Files and directories in", directory_path)
     for item in file_list:
         print(item)

Removing a Directories

  • You can remove a directory using os.rmdir()
import os
     
     # Specify the directory path to be removed
     directory_path = "/path/to/your/directory"
     
     # Remove the directory
     try:
         os.rmdir(directory_path)
         print(f"Directory '{directory_path}' removed successfully.")
     except OSError as e:
         print(f"Error: {e}")

 

Using os.path.join() for Platform Independence

  • When working with paths, it's a good practice to use os.path.join() to join directory and file names, making your code platform-independent
import os
     
     # Specify directory and file names
     base_directory = "/path/to/your"
     subdirectory = "example_directory"
     file_name = "example_file.txt"
     
     # Use os.path.join() to create the full path
     full_path = os.path.join(base_directory, subdirectory, file_name)
     
     # Print the full path
     print("Full path:", full_path)
