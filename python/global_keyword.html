Global Keyword

  • The global keyword is used to indicate that a variable declared inside a function should refer to the global variable of the same name, rather than creating a new local variable with the same name.
  • Without the global keyword, if you try to modify a variable inside a function with the same name as a global variable, Python will create a new local variable within the function, and any modifications will only affect the local scope.
