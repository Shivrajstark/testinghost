List 

  • Lists are a fundamental and powerful data structure in Python, and they are widely used for various purposes, from simple storage of elements to complex data manipulation and processing.
  • A list is a versatile and mutable data structure that can hold an ordered collection of items.
  • Lists are defined by enclosing the elements in square brackets [ ] and separating them with commas.

Key features of lists:

  • Ordered
  • Mutable
  • Dynamic
  • Common Opreations
  • Length

 

Ordered

  • Lists maintain the order of elements as they are inserted.
  • You can access elements by their index.
# Ordered List Example
     
     # Define an ordered list
     numbers = [1, 2, 3, 4, 5]
     
     # Print the ordered list
     print("Ordered List:", numbers)

Mutable

  • Lists can be modified after creation. You can change, add, or remove elements.
# Define a mutable list
     numbers = [1, 2, 3, 4, 5]  # No output, list creation
     
     # Print the original list
     print("Original List:", numbers)  # Output: Original List: [1, 2, 3, 4, 5]
     
     # Modify the list (add an element)
     numbers.append(6)  # No output, list modification
     
     # Print the modified list
     print("Modified List:", numbers)  # Output: Modified List: [1, 2, 3, 4, 5, 6]

Dynamic

  • Lists in Python can grow or shrink in size dynamically.
# Dynamic List Example
     
     # Create an empty dynamic list
     dynamic_list = []  # No output, empty list creation
     
     # Get user input to populate the list
     num_elements = int(input("Enter the number of elements: "))
     
     for i in range(num_elements):
         element = input(f"Enter element {i + 1}: ")
         dynamic_list.append(element)  # No output, list modification
     
     # Print the dynamic list
     print("Dynamic List:", dynamic_list)  # Output: Dynamic List: [element1, element2, ..., elementN]

Common Operations

  • Various operations can be performed on lists, such as slicing, concatenation, and repetition.
# Create a sample list
     my_list = [1, 2, 3, 4, 5]  # No output, list creation
     
     # Accessing elements and modifying
     first_element = my_list[0]  # Output: Accessing the first element
     my_list[2] = 10  # No output, modifying an element
     
     # Appending an element
     my_list.append(6)  # Output: Appending 6 to the list
     
     # Removing and getting length
     removed_element = my_list.pop(3)  # Output: Removing the element at index 3
     list_length = len(my_list)  # Output: Getting the length of the list
     
     # Checking membership
     element_in_list = 6 in my_list  # Output: Checking if 6 is in the list
     
     # Printing the results
     print("Original List:", my_list)  # Output: Original List: [1, 2, 10, 4, 5, 6]
     print("First Element:", first_element)  # Output: First Element: 1
     print("Removed Element:", removed_element)  # Output: Removed Element: 4
     print("List Length:", list_length)  # Output: List Length: 5
     print("Is 6 in List:", element_in_list)  # Output: Is 6 in List: True

Length

  • You can find the number of elements in a list using the len() function.
# Length of a List Example
     
     # Create a sample list
     my_list = [1, 2, 3, 4, 5]
     
     # Find the length of the list
     list_length = len(my_list)  # No output, finding the length of the list
     
     # Print the result
     print("List Length:", list_length)  # Output: List Length: 5
