Dictionary

  • Dictionaries are widely used in Python for tasks that involve mapping keys to values, such as storing configurations, representing JSON-like structures, or counting occurrences of elements.
  • A dictionary is a data structure that stores a collection of key-value pairs. It is defined by enclosing the key-value pairs within curly braces { } .
  • Dictionaries are also known as associative arrays, hash maps, or hash tables in other programming languages.

Key features of Dictionary:

  • Unordered
  • Mutable
  • Dynamic
  • Common Opreations
  • Key and Values

 

Unordered

  • Unlike lists, dictionaries do not maintain any order for the elements.
  • Elements are accessed using keys rather than indices.
# Creating an unordered dictionary
     my_dict = {"name": "Alice", "age": 25, "city": "Wonderland"}
     
     # Accessing values by key
     print("Name:", my_dict["name"])
     # Output: Name: Alice
     
     # Modifying values by key
     my_dict["age"] = 26
     print("Updated dictionary:", my_dict)
     # Output: Updated dictionary: {'name': 'Alice', 'age': 26, 'city': 'Wonderland'}
     
     # Adding a new key-value pair
     my_dict["gender"] = "Female"
     print("Dictionary after adding gender:", my_dict)
     # Output: Dictionary after adding gender: {'name': 'Alice', 'age': 26, 'city': 'Wonderland', 'gender': 'Female'}
     
     # Removing a key-value pair
     removed_value = my_dict.pop("city")
     print("Dictionary after removing city:", my_dict)
     print("Removed city value:", removed_value)
     # Output: Dictionary after removing city: {'name': 'Alice', 'age': 26, 'gender': 'Female'}
     # Output: Removed city value: Wonderland
     
     # Checking if a key is present in the dictionary
     is_age_present = "age" in my_dict
     print("Is 'age' present in the dictionary?", is_age_present)
     # Output: Is 'age' present in the dictionary? True
     
     # Iterating through key-value pairs
     print("Iterating through key-value pairs:")
     for key, value in my_dict.items():
         print(f"{key}: {value}")
     # Output:
     # name: Alice
     # age: 26
     # gender: Female

Mutable

  • Dictionaries can be modified after creation. You can add, modify, or remove key-value pairs.
# Creating a mutable dictionary
     my_dict = {"name": "Alice", "age": 25, "city": "Wonderland"}
     print("Original Dictionary:", my_dict)
     # Output: Original Dictionary: {'name': 'Alice', 'age': 25, 'city': 'Wonderland'}
     
     # Modifying values in the dictionary
     my_dict["age"] = 26
     my_dict["gender"] = "Female"
     print("Modified Dictionary:", my_dict)
     # Output: Modified Dictionary: {'name': 'Alice', 'age': 26, 'city': 'Wonderland', 'gender': 'Female'}
     
     # Removing a key-value pair from the dictionary
     removed_value = my_dict.pop("city")
     print("Dictionary after removing 'city':", my_dict)
     print("Removed city value:", removed_value)
     # Output: Dictionary after removing 'city': {'name': 'Alice', 'age': 26, 'gender': 'Female'}
     # Output: Removed city value: Wonderland

Dynamic

  • Like lists, dictionaries can grow or shrink in size dynamically.
# Creating an empty dictionary
     my_dict = {}
     print("Empty Dictionary:", my_dict)
     # Output: Empty Dictionary: {}
     
     # Dynamically adding key-value pairs
     my_dict["name"] = "Alice"
     my_dict["age"] = 25
     print("Dictionary after adding key-value pairs:", my_dict)
     # Output: Dictionary after adding key-value pairs: {'name': 'Alice', 'age': 25}
     
     # Dynamically modifying values
     my_dict["age"] = 26
     print("Dictionary after modifying 'age':", my_dict)
     # Output: Dictionary after modifying 'age': {'name': 'Alice', 'age': 26}
     
     # Dynamically removing a key-value pair
     removed_value = my_dict.pop("name")
     print("Dictionary after removing 'name':", my_dict)
     print("Removed name value:", removed_value)
     # Output: Dictionary after removing 'name': {'age': 26}
     # Output: Removed name value: Alice

Common Operations

  • Various operations can be performed on dictionaries, such as checking if a key exists, iterating through key-value pairs, and copying dictionaries.

 

Key and Values

  • Dictionaries have keys and values. Keys must be unique within a dictionary, and they are typically strings or numbers. Values can be of any data type.
# Creating a dictionary
     my_dict = {"name": "Alice", "age": 25, "city": "Wonderland"}
     print("Original Dictionary:", my_dict)
     
     # Getting keys
     keys_list = list(my_dict.keys())
     print("Keys:", keys_list)
     
     # Getting values
     values_list = list(my_dict.values())
     print("Values:", values_list)
