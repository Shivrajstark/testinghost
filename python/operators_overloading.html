Object Oriented Programming

Operators Overloading

  • Operator overloading in Python allows you to define how operators behave for objects of a class.
  • It enables you to use standard operators (+, -, *, /, etc.) with instances of your custom classes. This can make your code more readable and expressive.

Some Operators Overloading are : 

  • Overloading Binary Operators
# Simple Overloading of Binary Operator Example
     
     class Number:
         def __init__(self, value):
             self.value = value
     
         def __add__(self, other):
             if isinstance(other, Number):
                 return Number(self.value + other.value)
             else:
                 raise ValueError("Unsupported operand type for +")
     
         def __str__(self):
             return str(self.value)
     
     # Create instances of Number
     num1 = Number(5)
     num2 = Number(3)
     
     # Use the overloaded + operator
     result = num1 + num2
     print("Result:", result)  # Output: Result: 8
  • Overloading Unary Operators
# Overloading Unary Operator Example
     
     class Number:
         def __init__(self, value):
             self.value = value
     
         def __neg__(self):
             """Overloading the negation (-) operator."""
             return Number(-self.value)
     
         def __str__(self):
             """String representation of the Number."""
             return str(self.value)
     
     # Create an instance of Number
     num = Number(5)
     
     # Use the overloaded - operator
     negated_num = -num
     print("Original Number:", num)  # Output: Original Number: 5
     print("Negated Number:", negated_num)  # Output: Negated Number: -5
  • Overloading Comparison Operators
# Overloading Comparison Operators Example
     
     class Person:
         def __init__(self, age):
             self.age = age
     
         def __eq__(self, other):
             """Overloading the equality (==) operator."""
             if isinstance(other, Person):
                 return self.age == other.age
             else:
                 return False
     
         def __lt__(self, other):
             """Overloading the less than (<) operator."""
             if isinstance(other, Person):
                 return self.age < other.age
             else:
                 raise ValueError("Unsupported operand type for <")
     
     # Create instances of Person
     person1 = Person(25)
     person2 = Person(30)
     person3 = Person(25)
     
     # Use the overloaded == and < operators
     equal_result = person1 == person3
     lt_result = person1 < person2
     
     print("Equality Result:", equal_result)  # Output: Equality Result: True
     print("Less Than Result:", lt_result)  # Output: Less Than Result: True
