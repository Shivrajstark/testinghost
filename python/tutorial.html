Introduction to Python:

  • Python is an open source programming language. Nowadays  python comes under the category of top 3-4 languages
  • Python was created by Guido van rossum in 1991.
  • Python is a general purpose language by general purpose we mean we can use python anywhere there is no specific domain for it.

Why Python?

There are the following reasons why Python is used:-

  • Simple and easy to learn From a student point of view, python became popular among students who focus on solutions rather than syntax.
  • Platform-independent Python can run on any platform like Linux and Windows, so platform independence is also the main reason to use Python.
  • Free and open source is freely available, also source code can be easily downloaded.
  • The rich library supports up to 1 lakh 40 thousand languages .simple we need to import the code there is no need to write full code from Scratch.
