Break and Continue Statements 

  •   Break :  ' break ' Terminates the loop it is in, and the program continues with the next statement outside the loop.
  •   Continue :  ' continue ' Skips the rest of the code inside the loop for the current iteration and goes to the next iteration.
# Example using break in a loop
     print("Example using break:")
     for i in range(5):
         if i == 3:
             # When i is 3, break out of the loop
             break
         print(i)
     # Output: 0
     #         1
     #         2
     
     # Example using continue in a loop
     print("\nExample using continue:")
     for j in range(5):
         if j == 2:
             # When j is 2, skip the rest of the loop and continue with the next iteration
             continue
         print(j)
     # Output: 0
     #         1
     #         3
     #         4
